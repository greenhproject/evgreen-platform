# Análisis de Competencia: Vatio EV (Colombia)

## Resumen
Vatio EV es una aplicación de carga de vehículos eléctricos operando en Colombia. El análisis se basa en capturas de pantalla de su aplicación móvil.

## Diseño Visual

### Tema y Colores
- **Tema oscuro** como base principal
- **Verde (#4CAF50 aproximado)** como color de acento
- Fondo negro/gris oscuro para todas las pantallas
- Texto blanco/gris claro para contraste

### Tipografía
- Fuente sans-serif moderna
- Jerarquía clara entre títulos y contenido

## Pantalla Principal (Mapa)

### Elementos de UI
1. **Barra de búsqueda superior**
   - Placeholder: "Buscar estaciones cercanas"
   - Icono de lupa a la izquierda
   - Avatar de usuario a la derecha

2. **Mapa interactivo**
   - Estilo dark mode
   - Marcador de ubicación del usuario (punto azul)
   - Marcadores de estaciones (no visibles en la captura)

3. **Botones flotantes (derecha)**
   - Refrescar mapa
   - Filtros
   - Centrar en ubicación

4. **Barra inferior de acciones rápidas**
   - Descuento (cupones/promociones)
   - Tarjeta RFID
   - **Botón central verde grande** - Escanear QR
   - Saldo actual ($0)
   - Vehículo registrado

## Menú Lateral (Drawer)

### Estructura
- Avatar de usuario (circular, con silueta)
- Nombre del usuario
- Lista de opciones con iconos verdes:
  - Mi cuenta
  - Billetera
  - Historial de carga
  - Favoritos
  - VIP (programa de membresía)
  - Comentarios
  - Contáctanos
  - Ajustes
  - Ayuda

## Gestión de Vehículos

### Características
- Header con botón atrás y "Añadir"
- Estado vacío con ilustración de auto
- Texto "Sin contenido"

## Escáner de Código QR

### Elementos
- Cámara a pantalla completa
- Marco de escaneo con esquinas redondeadas
- Línea de escaneo animada (verde)
- Instrucciones: "Coloque un código de barras en el interior del rectángulo del visor para escanear"
- Botones inferiores:
  - Encender luz (flash)
  - Entrada manual (código numérico)
- Botón de galería (esquina superior derecha)

## Billetera

### Sección de Recargas
- Card verde con saldo actual prominente
- Botón "Historial" en la esquina

### Montos Predefinidos
- $20,000 COP
- $40,000 COP
- $100,000 COP
- Cada uno muestra el precio real

### Campo de Monto Personalizado
- Input con símbolo de peso
- Valor editable

### Botón de Acción
- "Recargar Ahora" - botón verde grande

### Tarjetas Guardadas
- Sección "Mis tarjetas(0)"
- Botón "Agregar Nueva Tarjeta"

---

## Oportunidades de Mejora para Green EV

1. **Diseño más moderno** - Usar glassmorphism y gradientes sutiles
2. **Animaciones fluidas** - Transiciones suaves entre pantallas
3. **Información en tiempo real** - Mostrar estado de carga en el mapa
4. **Reservas** - Funcionalidad que Vatio no muestra prominentemente
5. **Dashboard de inversionista** - Diferenciador clave
6. **Soporte en vivo** - Chat integrado
7. **Programa de suscripción** - Más visible que el "VIP" de Vatio
8. **Estimación de tiempo** - Mostrar tiempo estimado de carga disponible

Información General

La API de EVGreen permite integrar tu aplicación con nuestra plataforma de carga de vehículos eléctricos. Consulta estaciones, transacciones, envía comandos remotos y recibe notificaciones en tiempo real.

Base URL

https://www.evgreen.lat/api/v1

Formato de Respuesta

Todas las respuestas son JSON con la siguiente estructura:

{
  "success": true,
  "data": { ... },
  "pagination": {
    "page": 1,
    "limit": 20,
    "total": 5
  }
}

🔐 Autenticación

Todas las solicitudes requieren una API Key válida. Puedes generar API Keys desde el panel de administración de EVGreen.

Cómo autenticarse

Incluye tu API Key en el header X-API-Key o como query param api_key:

# Opción 1: Header (recomendado)
curl -H "X-API-Key: evg_tu_api_key_aqui" \
  https://www.evgreen.lat/api/v1/stations

# Opción 2: Query parameter
curl "https://www.evgreen.lat/api/v1/stations?api_key=evg_tu_api_key_aqui"

Límites de Uso

LímiteValorDescripción
Rate Limit100 req/minMáximo 100 solicitudes por minuto por API Key
Paginaciónmax 100Máximo 100 registros por página

Generar API Key

Para obtener una API Key, contacta al administrador de EVGreen o genera una desde el panel admin en Configuración → API Keys.

🎮 Comandos Remotos

Envía comandos OCPP a las estaciones de carga. Solo disponible para API Keys con rol admin/staff.

POST /stations/:id/start Iniciar carga remota

Body (JSON)

CampoTipoRequeridoDescripción
connectorIdnumberSíID del conector
idTagstringSíTag RFID o ID del usuario (formato: EV-XXXXXX)

Ejemplo

curl -X POST -H "X-API-Key: evg_xxx" \
  -H "Content-Type: application/json" \
  -d '{"connectorId": 1, "idTag": "EV-001234"}' \
  "https://www.evgreen.lat/api/v1/stations/1/start"

Respuesta

{
  "success": true,
  "data": {
    "stationId": 1,
    "connectorId": 1,
    "idTag": "EV-001234",
    "commandResult": { "status": "Accepted" },
    "timestamp": "2026-05-04T15:00:00Z"
  }
}
POST /stations/:id/stop Detener carga remota

Body (JSON)

CampoTipoRequeridoDescripción
transactionIdnumberSíID de la transacción activa

Ejemplo

curl -X POST -H "X-API-Key: evg_xxx" \
  -H "Content-Type: application/json" \
  -d '{"transactionId": 42}' \
  "https://www.evgreen.lat/api/v1/stations/1/stop"

🔔 Webhooks

Recibe notificaciones en tiempo real cuando ocurren eventos en la plataforma.

GET /webhooks Listar webhooks configurados

Respuesta

{
  "success": true,
  "data": [
    {
      "id": 1,
      "url": "https://tu-app.com/webhook",
      "events": ["charging.started", "charging.completed"],
      "isActive": true,
      "lastTriggeredAt": "2026-05-04T14:00:00Z"
    }
  ]
}
POST /webhooks Registrar un nuevo webhook

Body (JSON)

CampoTipoRequeridoDescripción
urlstringSíURL que recibirá los eventos (HTTPS)
eventsstring[]SíLista de eventos a suscribir

Eventos Disponibles

EventoDescripción
charging.startedSe inició una sesión de carga
charging.completedSe completó una sesión de carga
charging.failedFalló una sesión de carga
station.onlineEstación se conectó
station.offlineEstación se desconectó
alert.createdSe generó una alerta

Ejemplo

curl -X POST -H "X-API-Key: evg_xxx" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://tu-app.com/evgreen-webhook",
    "events": ["charging.started", "charging.completed"]
  }' \
  "https://www.evgreen.lat/api/v1/webhooks"

❌ Códigos de Error

La API usa códigos HTTP estándar y devuelve errores en formato JSON:

{
  "error": "ERROR_CODE",
  "message": "Descripción legible del error"
}

🛠️ Ejemplos de Integración

Python

import requests

API_KEY = "evg_tu_api_key"
BASE_URL = "https://www.evgreen.lat/api/v1"
headers = {"X-API-Key": API_KEY}

# Listar estaciones
response = requests.get(f"{BASE_URL}/stations", headers=headers)
stations = response.json()["data"]

# Ver estado en tiempo real
status = requests.get(f"{BASE_URL}/stations/1/status", headers=headers)
print(status.json()["data"]["connectionStatus"])

# Iniciar carga remota
payload = {"connectorId": 1, "idTag": "EV-001234"}
result = requests.post(f"{BASE_URL}/stations/1/start", 
                       json=payload, headers=headers)
print(result.json())

JavaScript / Node.js

const API_KEY = "evg_tu_api_key";
const BASE_URL = "https://www.evgreen.lat/api/v1";

// Listar estaciones
const response = await fetch(`${BASE_URL}/stations`, {
  headers: { "X-API-Key": API_KEY }
});
const { data: stations } = await response.json();

// Ver estadísticas
const stats = await fetch(`${BASE_URL}/stats/overview`, {
  headers: { "X-API-Key": API_KEY }
}).then(r => r.json());

console.log(`Energía total: ${stats.data.last30Days.totalEnergyKwh} kWh`);
